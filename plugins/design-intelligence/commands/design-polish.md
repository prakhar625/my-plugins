# /design-polish

Final polish pass before handoff.

## Prerequisites
- Design should be functionally complete
- Audit should pass (or issues acknowledged)

## Workflow

### Step 1 — Micro-Refinements
Review and refine:
- Pixel-level alignment
- Spacing consistency
- Typography fine-tuning
- Color harmony

### Step 2 — Edge Cases
Check for:
- Empty states
- Error states
- Loading states
- Long content handling
- Truncation behavior

### Step 3 — Delight Details
Consider additions:
- Subtle animations (if appropriate)
- Micro-interactions
- Thoughtful defaults
- Helpful empty states

### Step 4 — Final Verification
- [ ] All orphaned values resolved
- [ ] All states documented
- [ ] Anti-slop check passes
- [ ] Accessibility verified

## Output

- Polish changes documented
- Before/after comparisons
- Ready for handoff
